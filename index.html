Hexo + Github Pages 搭建博客

发表于 2023-08-31 更新于 2023-09-01

GitHub Pages 是GitHub推出的静态网站托管服务,而 Hexo 是一个快速、简洁且高效的静态博客框架.可以把GitHub Pages和Hexo结合,搭建一个简单、漂亮、免费的博客.本文将介绍如何安装配置 GitHub Pages 和 Hexo Next 主题.

1. 本地搭建Hexo环境

1.1 首先安装nodejs和git,官网下载Windows平台安装包,直接点击”下一步”完成安装.

1.2 创建hexo存放目录,并初始化hexo.

1.3 安装next主题,并配置站点配置文件.

1.4 新建文章并启动hexo

2. 创建GitHub Pages

2.1 创建GitHub Pages时,仓库名称格式为：[github username].github.io

3. 部署博客到GitHub

3.1 生成ssh密钥,将私钥放在”c:\Users\Administrator\.ssh目录中”,公钥上传GitHub.

公钥上传GitHub方法：

登陆Github->左键”头像”->”Settings”→”SSH and GPG kyes”→”New SSH key”,依次填写”Title”,将上面生成的公钥内容复制到”Key”下面的方框,最后点击”Add SSH key”完成添加.

3.2 配置本地账户

3.3 部署blog到GitHub,需要在”E:\blog\_config.yml”中增加”deploy”配置项,还需要安装部署插件.

4. 访问博客

https://ibiubiu.github.io

5. 部署总结

免费的简易博客到此就部署成功.过程非常简单,也存在一个很重要的问题。那就是hexo deploy自动部署，提交到仓库的文件是public文件夹。并没有提交hexo环境，如果想跨平台,多终端使用的话，还是需要重新安装hexo环境。还是比较麻烦。有没有比较好的解决方法呢？

可以通过GitHub Actions自动部署或者Gitlab自动部署解决此问题.下一篇博客继续.
